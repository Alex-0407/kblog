# kblog

A simple blog build with Koa2. Markdown and syntax highlight supported.

## Requirements

1. Node.js 6.x
2. MongoDB
3. Nginx

## Deploy

1. Clone project: git clone https://github.com/hojas/kblog.git
2. Install npm packages: cd kblog && npm install
3. Build app: npm run build
4. Run app: npm start
